# StabLab Processor 使用指南

## 1. 软件简介

StabLab Processor 是一款专为太阳能电池（Solar Cell）,尤其是新型的有机、钙钛矿等太阳能电池稳定性测试设计的数据分析与可视化软件。它能够快速解析多通道测试日志，生成填充因子 (FF)、开路电压 (Voc)、短路电流 (Isc)、最大功率 (Pmax) 以及累积发电量的可视化曲线，并支持灵活的多通道数据分组与一键导出。

## 2. 核心功能

- **多维度数据可视化**：支持 FF, Voc, Isc, Pmax 及累积发电量曲线的无缝切换。
- **灵活的数据分组**：支持通过导入配置文件，批量对测试通道进行分组对比与统一着色。
- **智能数据处理**：内置微积分算法，自动根据功率和时间戳计算器件的累积发电量 (mWh)。
- **数据导出**：支持将当前视图下处理、清洗后的数据导出为标准 CSV 格式，方便导入 Origin 等专业绘图软件。
- **离线运行**：支持打包为本地桌面应用程序，满足实验室断网环境下的安全使用需求。

## 3. 数据格式规范

### 3.1 测试数据文件 (Log Files)

软件支持导入标准的 CSV 或 TXT 格式测试日志。

- **文件命名要求**：文件名中必须包含通道号标识，格式为 `C` + `数字`（例如：`data_C001.txt`，`sample_C012.csv`）。软件会自动提取 `C001`、`C012` 作为该条曲线的通道名称。
- **文件内容要求**：数据必须以逗号分隔，且必须包含以下表头（区分大小写，必须包含括号内的单位）：
  - `Time`：时间戳（例如 `2023-10-01 12:00:00`）
  - `Isc(mA)`：短路电流
  - `Voc(V)`：开路电压
  - `Pmax(mW)`：最大功率
  - `FF(%)`：填充因子

**测试数据文件示例 (`data_C001.csv`)：**

```csv
Time,Isc(mA),Voc(V),Pmax(mW),FF(%)
2023-10-01 10:00:00,20.5,1.05,17.2,80.0
2023-10-01 10:05:00,20.4,1.04,17.0,79.8
2023-10-01 10:10:00,20.6,1.05,17.3,80.1
```

### 3.2 平铺测试数据文件 (Flat Format)
除了标准的按列排列的 CSV/TXT 文件外，软件现在导入无换行符的**连续平铺数据格式**，这常见于欧姆早期的稳定性数据软件。

*   **数据结构要求**：数据由空格或制表符（Tab）分隔，连续排列，无换行符。
*   **数据块要求**：每 10 个数据点作为一个完整的数据块（代表一次测量记录）。
*   **支持的字段顺序**：
    1. `Timestamp`：时间戳（支持 `YYYYMMDD_HHMMSS` 格式，如 `20251020_001417`，或标准日期格式）。
    2. `Channel`：通道号（如 `1`，软件会自动转换为 `C001`）。
    3. `Voc(V)`：开路电压。
    4. `Jsc(A)`：短路电流密度（软件会自动取绝对值并转换为 `mA` 单位的 `Isc`）。
    5. `Vmp(V)`：最大功率点电压。
    6. `Jmp(A)`：最大功率点电流密度。
    7. `Pmp(W)`：最大功率（软件会自动转换为 `mW` 单位的 `Pmax`）。
    8. `FF`：填充因子。
    9. `Rs(ohm)`：串联电阻。
    10. `Rsh(ohm)`：并联电阻。
*   **多通道混合**：同一个文件中可以包含多个不同通道的数据，软件在解析时会自动将它们拆分为多条独立的曲线。


### 3.3 分组配置文件 (Cluster Config)

如果需要对多个通道进行分组对比（例如对比不同封装材料的衰减情况），可以导入分组配置文件。

- **文件格式**：CSV 格式（`.csv`）
- **表头要求**：
  - `Group`：分组名称（如：对照组、实验组A）。
  - `Color`：该分组在图表中的显示颜色。支持 HEX 颜色码（如 `#FF5733`）或标准英文颜色名（如 `blue`，`red`）。
  - `Channels`：该组包含的通道号。支持逗号分隔和连字符范围（例如 `"1-5, 8, 10-12"` 代表通道 1,2,3,4,5,8,10,11,12）。**注意**：如果包含逗号，整个字段需要用双引号包裹。

**分组配置文件示例 (`groups.csv`)：**

```csv
Group,Color,Channels
对照组,#FF5733,"1-3, 7"
实验组A,#33FF57,"4-6"
实验组B,blue,"8, 10-12"
```

## 4. 操作步骤

1. **导入数据**：在左侧边栏点击“上传文件”区域，选择一个或多个符合格式的测试数据文件（如 `C001.txt`，`C002.txt`）。
2. **切换视图**：在图表上方点击不同的参数标签（FF，Voc，Isc，Pmax，累积发电量），图表会实时更新并自动调整坐标轴范围。
3. **配置分组**：点击左侧边栏底部的“导入分组配置”按钮，上传准备好的 `groups.csv` 文件，图表线条将自动按组重新着色并归类，图例也会同步更新。
4. **导出数据**：点击界面右上角的“导出 CSV”按钮，即可将当前视图下处理好的数据（包括计算出的累积发电量等）保存到本地电脑。
